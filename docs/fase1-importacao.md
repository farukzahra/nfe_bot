# Fase 1 — Importação de Documentos Fiscais

## Objetivo

Permitir que o usuário importe arquivos fiscais no sistema.

Formatos aceitos inicialmente:

- XML de NF-e
- XML de NFC-e
- Arquivo ZIP contendo múltiplos XMLs

## Prioridade Técnica

A prioridade deve ser o XML, porque ele já contém os dados estruturados de forma confiável.

Ordem recomendada:

1. XML de NF-e/NFC-e
2. ZIP com XMLs
