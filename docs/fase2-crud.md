# Fase 2 — Tela de CRUD dos Imports

Depois da importação, o sistema deve ter uma tela simples para visualizar e corrigir os dados importados.

## Funcionalidades

- Listar documentos importados
- Filtrar por:
  - Entrada
  - Saída
  - Data
  - CNPJ
  - Cliente
  - Fornecedor
  - Produto
  - Valor
- Visualizar detalhes da nota
- Visualizar itens da nota
- Visualizar impostos
- Ver XML original
- Reprocessar documento
- Excluir importação
- Corrigir classificação manualmente, se necessário

## Organização Visual

A tela deve sempre separar:

- Entradas
- Saídas

Exemplo:

- Aba 1: Entradas
- Aba 2: Saídas
- Aba 3: Todos
- Aba 4: Erros de importação
